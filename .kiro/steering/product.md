# Product Overview

**Noto** is a PDF annotation and note-taking application that allows users to:

- Upload and view PDF documents
- Select text within PDFs and create annotations with personal notes
- Manage annotations across multiple PDF documents
- View annotation previews and tooltips when hovering over highlighted text
- Organize notes and annotations in a dashboard interface

## Core Features

- **PDF Viewer**: Built with Syncfusion PDF Viewer component for robust PDF rendering
- **Text Selection & Annotation**: Users can select text and attach notes to specific coordinates
- **User Authentication**: Clerk-based authentication system
- **Dashboard Interface**: Clean, modern UI for managing PDFs and annotations
- **Real-time State Management**: Redux Toolkit for managing application state
- **Responsive Design**: Tailwind CSS with mobile-friendly layouts

## Target Users

Researchers, students, and professionals who need to annotate PDF documents and organize their notes efficiently.
